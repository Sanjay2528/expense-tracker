## Problem Statement
Managing daily expenses can be challenging and often leads to poor budgeting due to a lack of proper tools. Manual tracking is time-consuming, prone to errors, and makes it difficult for users to analyze their spending patterns. The Expense Tracker project addresses this problem by providing a user-friendly Python application for recording, viewing, and categorizing personal expenses.

## Scope of the Project
- Console-based program for individual use
- Allows users to add, view, and categorize expenses into preset groups (Food, Travel, Shopping, Bills, Others)
- Computes total expenses and provides a summary by category
- Focuses on essential functionality for beginner programming and personal budgeting
- All data stored in memory (no database/file storage)

## Target Users
- Individuals wishing to manage and analyze their personal spending
- Students and beginners learning programming with practical examples
- Anyone wanting a simple, effective tool to gain better control over their finances

## High-Level Features
- Add an expense with amount and category
- View a list of all recorded expenses
- Get instant total expense calculation
- Category-wise summary of spending
- Simple menu-driven console interface for easy use
